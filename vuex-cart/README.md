![](https://github.com/baixue0111/vue-study/blob/master/vuex-cart/src/assets/20171114144112122.gif)

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
